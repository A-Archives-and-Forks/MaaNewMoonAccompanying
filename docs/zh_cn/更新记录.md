# 更新记录

## 版本格式说明

版本的格式一般为：v`迭代版本`-`大版本`-`小版本`

- 迭代版本：每次架构或技术更新、具有时期意义时会自增，`0`表示开发测试期版本
- 大版本：进入新开发周期、游戏大版本迭代时会自增
- 小版本：每次发布`Release`版本时会自增

## 版本主题

| 大版本  |   主要内容   |
| :----: | :----------: |
|  v0.1  |  可行性测试  |
|  v0.2  | 框架与UI适配 |  
|  v0.3  |   日常功能   |  
|  v0.4  |   周常功能   |
|  v1.0  |   实用小功能   |
|  v1.1 (now)  |   细节扩展   |
